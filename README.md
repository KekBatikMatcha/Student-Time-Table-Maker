# Timetable Maker

A simple and interactive student timetable maker built with HTML, CSS, JavaScript, and PHP.

## Files
- `timetable.html` – Main interface and timetable editor
- `timetable.php` – Server-side support (if used)

## Features
- Create weekly class schedules
- Custom days and time ranges
- Color and layout customization
- Print-ready timetable
- Export timetable as high-quality image (PNG)

## Usage
1. Open `timetable.html` in a browser  
2. Create and customize your timetable  
3. Print or export when ready  

## Purpose
This project was created for learning, portfolio, and practical student use.
